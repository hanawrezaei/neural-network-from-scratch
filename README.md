# Neural Network From Scratch

A clean, object-oriented implementation of a single neuron built from scratch using **NumPy** to build a strong practical foundation in deep learning.

## Features
- **Object-Oriented Design:** Clean and structured `Neuron` class with standard `forward` and `backward` steps.
- **Forward Pass:** Implements matrix dot product combined with a Sigmoid activation function.
- **Backward Pass:** Hand-calculated backpropagation using the chain rule to compute gradients and optimize both weights ($W$) and bias ($b$).
- **Live Training Simulation:** Demonstrates the loss converging over epochs dynamically inside Jupyter Notebook.

## How to Run
1. Download the notebook file: `single_neuron.ipynb`
2. Open it using Jupyter Notebook or VS Code to see the live execution.
